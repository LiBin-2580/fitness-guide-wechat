## 二、数据表设计

### 2.1 用户表 (users)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 主键 |
| openid | VARCHAR(64) | UNIQUE, NOT NULL | 微信openid |
| nickname | VARCHAR(50) | DEFAULT '' | 用户昵称 |
| avatar | VARCHAR(255) | DEFAULT '' | 头像URL |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2.2 训练计划表 (plans)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 主键 |
| user_id | INT | FOREIGN KEY, NOT NULL | 用户ID |
| name | VARCHAR(100) | NOT NULL | 计划名称 |
| description | TEXT | | 计划描述 |
| difficulty | ENUM('easy','medium','hard') | DEFAULT 'medium' | 难度等级 |
| duration_weeks | INT | DEFAULT 4 | 计划周期（周） |
| status | TINYINT | DEFAULT 1 | 状态：1启用 0停用 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2.3 训练动作表 (exercises)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 主键 |
| plan_id | INT | FOREIGN KEY, NOT NULL | 计划ID |
| day_of_week | TINYINT | NOT NULL | 星期几（1-7） |
| name | VARCHAR(100) | NOT NULL | 动作名称 |
| sets | INT | DEFAULT 3 | 组数 |
| reps | VARCHAR(20) | DEFAULT '12' | 次数（如"10-12"） |
| rest_seconds | INT | DEFAULT 60 | 组间休息（秒） |
| sort_order | INT | DEFAULT 0 | 排序序号 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 2.4 训练记录表 (records)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 主键 |
| user_id | INT | FOREIGN KEY, NOT NULL | 用户ID |
| plan_id | INT | FOREIGN KEY | 计划ID |
| exercise_id | INT | FOREIGN KEY | 动作ID |
| sets_completed | INT | DEFAULT 0 | 完成组数 |
| reps_completed | VARCHAR(20) | | 完成次数 |
| weight_used | DECIMAL(6,2) | DEFAULT 0 | 使用重量(kg) |
| duration_seconds | INT | DEFAULT 0 | 训练时长(秒) |
| completed_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 完成时间 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 2.5 身体数据表 (body_metrics)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 主键 |
| user_id | INT | FOREIGN KEY, NOT NULL | 用户ID |
| weight | DECIMAL(5,2) | | 体重(kg) |
| body_fat_rate | DECIMAL(5,2) | | 体脂率(%) |
| muscle_mass | DECIMAL(5,2) | | 肌肉量(kg) |
| bmi | DECIMAL(5,2) | | BMI指数 |
| notes | VARCHAR(200) | | 备注 |
| record_date | DATE | NOT NULL | 记录日期 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

## 三、索引设计

```sql
-- 用户表索引
CREATE UNIQUE INDEX idx_users_openid ON users(openid);

-- 计划表索引
CREATE INDEX idx_plans_user_id ON plans(user_id);

-- 动作表索引
CREATE INDEX idx_exercises_plan_id ON exercises(plan_id);
CREATE INDEX idx_exercises_day ON exercises(plan_id, day_of_week);

-- 记录表索引
CREATE INDEX idx_records_user_id ON records(user_id);
CREATE INDEX idx_records_date ON records(user_id, completed_at);

-- 身体数据索引
CREATE INDEX idx_metrics_user_id ON body_metrics(user_id);
CREATE INDEX idx_metrics_date ON body_metrics(user_id, record_date);
```
