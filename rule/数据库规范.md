# 数据库规范

> MySQL数据库开发规范

---

## 一、命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 表名 | snake_case，复数 | `users`、`training_records` |
| 字段名 | snake_case | `user_id`、`created_at` |
| 主键 | id | `id` |
| 外键 | 表名单数_id | `user_id`、`plan_id` |
| 时间字段 | xxx_at | `created_at`、`updated_at` |
| 索引 | idx_表名_字段 | `idx_users_openid` |

---

## 二、表设计规范

### 必备字段

每张表必须包含以下字段：

```sql
id          INT           PRIMARY KEY AUTO_INCREMENT  -- 主键
created_at  DATETIME      DEFAULT CURRENT_TIMESTAMP   -- 创建时间
updated_at  DATETIME      DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP  -- 更新时间
```

### 字段类型选择

| 场景 | 推荐类型 | 示例 |
|------|----------|------|
| 主键/外键 | INT | `id INT` |
| 短文本 | VARCHAR | `name VARCHAR(100)` |
| 长文本 | TEXT | `description TEXT` |
| 金额/精确小数 | DECIMAL | `weight DECIMAL(5,2)` |
| 布尔值 | TINYINT | `status TINYINT DEFAULT 1` |
| 枚举 | ENUM | `difficulty ENUM('easy','medium','hard')` |
| 日期 | DATE | `record_date DATE` |
| 时间 | DATETIME | `created_at DATETIME` |

## 三、SQL编写规范

### 3.1 查询规范

```sql
-- ✅ 明确指定字段，不用 *
SELECT id, name, created_at FROM plans WHERE user_id = 1;

-- ✅ 大量数据分页
SELECT * FROM records LIMIT 10 OFFSET 0;

-- ✅ 使用参数化查询（防SQL注入）
-- 代码中：db.query('SELECT * FROM users WHERE id = ?', [userId])

-- ❌ 避免拼接SQL
-- SELECT * FROM users WHERE id = '${userId}'
```

### 3.2 索引规范

```sql
-- ✅ 常用查询字段加索引
CREATE INDEX idx_plans_user_id ON plans(user_id);

-- ✅ 组合查询用联合索引
CREATE INDEX idx_records_user_date ON records(user_id, completed_at);

-- ✅ 唯一字段加唯一索引
CREATE UNIQUE INDEX idx_users_openid ON users(openid);
```

### 3.3 事务处理

```javascript
// 需要事务的场景：多个表同时操作
async function createPlanWithExercises(planData, exercises) {
  const conn = await db.getConnection();
  try {
    await conn.beginTransaction();
    
    const [planResult] = await conn.query('INSERT INTO plans SET ?', [planData]);
    const planId = planResult.insertId;
    
    for (const exercise of exercises) {
      await conn.query('INSERT INTO exercises SET ?', [{ ...exercise, plan_id: planId }]);
    }
    
    await conn.commit();
    return planId;
  } catch (err) {
    await conn.rollback();
    throw err;
  } finally {
    conn.release();
  }
}
```

## 四、连接配置

```javascript
// config/db.js
const mysql = require('mysql2/promise');

const pool = mysql.createPool({
  host: process.env.DB_HOST || 'localhost',
  port: process.env.DB_PORT || 3306,
  user: process.env.DB_USER || 'root',
  password: process.env.DB_PASSWORD || '',
  database: process.env.DB_NAME || 'fitness_guide',
  waitForConnections: true,
  connectionLimit: 10,
  queueLimit: 0,
});

module.exports = pool;
```
